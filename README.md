## Geral

### Descrição

Informações básicas e funcionalidade:

- Layout responsivo a diferentes telas e dispositivos;
- Horário em tempo real;
- Adicionar uma tarefa;
- Concluir, modificar e excluir a tarefa;
- Guardar a tarefa no navegador.
